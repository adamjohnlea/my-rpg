# Prototype: Lore as Power ("Knowledge is Power")

**Hypothesis tested:** Does lore-as-power discovery — gather lore while venturing (from
receptive peoples and sites; soldiers/barbarians refuse), complete a themed set → gain a
thematic mechanical perk — pull the player forward into exploration and feel rewarding, with
**no deduction**?

**PIVOT chain:** `unclaimed-valley-concept` (deduction → PIVOT) → this (lore-as-power).

**How to run:** Engine path (Phaser 4 / Vite). From repo root:
`npx vite prototypes/lore-as-power-concept --port 5182`
(or via `.claude/launch.json` → `lore-as-power-proto`). Open the served page.

**Status:** Concluded — 2026-06-26.

**Verdict:** **PROCEED** (hypothesis CONFIRMED). The lore→thematic-perk loop pulls the player
forward (chase the fire lore before the wildfire); the wildfire bouncing off Fireward makes
knowledge visibly = power. Social, faction-gated encounters (wanderer, soldier rebuff) were
the emotional high point. Caveat (prototype-only): fixed/labeled destinations felt flat — the
real game must make discovery **organic and non-linear across a large map**.

**Findings:** see [REPORT.md](REPORT.md).
