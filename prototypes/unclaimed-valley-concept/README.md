# Prototype: The Unclaimed Valley (Earned Discovery slice)

**Hypothesis tested:** Does uncovering a cataclysm by combining concrete *systemic* clues
(not lore text) produce an *earned* "aha!"? (Hearthfall "Earned Discovery" pillar.)

**How to run:** Open `prototype.html` in any browser (double-click — no server needed), or
serve via `.claude/launch.json` → `discovery-proto` (python static server, port 5181).

**Status:** Concluded — 2026-06-26.

**Verdict:** **PIVOT** (hypothesis PARTIALLY CONFIRMED). The deduction/"pick a conclusion"
step was the flat part and risks pulling the player out of the game; knowledge acquired from
a living person (the traveler) was the spark. Pivot to **lore-as-power**: gather lore through
venture/expand play, no deduction, each body of knowledge grants a thematic perk; some lore
gated behind receptive peoples (wanderers yes, soldiers/barbarians no).
